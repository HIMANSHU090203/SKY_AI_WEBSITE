# SKY Advanced Research

A premium AI company website featuring a stunning 3D black hole visualization using WebGL and Three.js.

## Features

- Realistic black hole simulation with gravitational lensing effect
- Interactive 3D starfield background
- Responsive design with modern UI elements
- WebGL-powered graphics for high-performance rendering

## Technologies Used

- HTML5/CSS3
- JavaScript
- Three.js for 3D graphics
- WebGL for GPU-accelerated rendering
- Custom GLSL shaders for the black hole effect

## Setup

Simply clone the repository and open index.html in a modern browser that supports WebGL.

```
git clone https://github.com/yourusername/SKYAdvancedResearch.git
cd SKYAdvancedResearch
```

## Browser Support

This website uses advanced WebGL features and requires a modern browser with WebGL support for optimal performance.
